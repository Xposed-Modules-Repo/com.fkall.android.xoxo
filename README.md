## AI 微信好友与群聊能力

- AI 白名单好友 / 群管理
- 支持为指定好友或群启用 AI 相关能力
- AI 自动回复，可按好友或群单独配置启用范围
- AI 接管 / 回复助手，支持生成候选回复、重新生成并手动发送
- AI 上下文与记忆配置
- 支持清理记忆
- 支持调整上下文内容和提示词
- AI 模型与接口配置
- 支持 API 地址配置
- 支持模型查询、配置保存和连通测试
- AI 图片生成与发送
- 支持图片生成接口、提示词、下载和本地发送
- AI 语音生成与发送
- 支持 MIMO2.5（推荐免费好用） 语音生成接口、TTS、音色样本、语音克隆和语音优化
- AI 视频生成与发送
- 支持视频生成接口、任务轮询和本地发送
- 群聊违规检测
- 违规统计
- 踢人处理
